User Stories

Guest User

  1. As a guest user, I need to search by keyword so I can find data I am interested in.
  2. As a guest user, I need my searches to also include embedded documents so I only have to search once for all relevant data.
  3. As a guest user, I need to be able to obtain search results from all pages nested within a web page so that I can save time researching a topic
  4. As a guest user, I want to be able to sort search results by custom criteria.
  5. As a guest user, I want to be able to sort search results by relevance.
  6. As a guest user, I want to be able to filter a list of search results by creation date so that I can omit search results that are too old to be relevant.
  7. As a guest user, I want to be able to save my search queries so that I can reference them later.
  8. As a guest user, I want to be able to remove queries from my saved searches so that I only maintain a relevant list of searches.
  9. As a guest user, I want to be able to sign into a custom account so that I will be able to reference my search queries.
  10. As a guest user, I want to have a document selected at random and displayed to me so I can understand what kind of data I can search.
  11. As a guest user, I want to see the most popular search terms so I can know what others are searching for.

Registered User

  1. As a registered user, I need to be able to save my search queries so that I can reference them later.
  2. As a registered user, I need to be able to remove queries from my saved searches so that I only maintain a relevant list of searches.
  3. As a registered user, I want to suggest content to the Mariana administrator so I can search it using Mariana.

Administrator

  1. As an administrator, I need to be able to sign into a custom account with privileges allowing me to interface with registered user accounts.
  2. As an administrator, I need to import new data into Mariana so my users can search it.
  3. As an administrator, I want to be able to reset the password of a registered user so that they will be able to regain access to their account.
  4. As an administrator, I want to be able to restore a registered userâ€™s deleted search queries upon request so they will be accessible again.
  5. As an administrator, I want to view reports on the searches that users are submitting so I can understand users' needs for search content.
  6. As an administrator, I want to see the most common searches users make so I can know what content is most popular.
  7. As an administrator, I want to restrict certain types of content to be searched only by registered users.

Application Tester

  1. As a test user, I need to test that the basic keyword search returns the same results given the same or similar input.
  2. As a test user, I need to test that all pages of the application load normally as expected.
  3. As a test user, I need to test that the REST API responds with the appropriate data in response to specific input.
  4. As a test user, I want to be able to automate my tests to make them more efficient.
  5. As a test user, I want to easily record the outcomes of all tests for later review.
